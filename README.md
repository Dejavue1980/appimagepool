### Build from source

- Download latest Flutter SDK (>=2.5.0 / dev version)
- Clone this repo and then 

```
# Download dependencies and Enable Linux support
$ flutter pub get; flutter config --enable-linux-desktop

# For Direct Testing
$ flutter run -v -d linux
```
